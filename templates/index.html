What is it about?

This event is about the sustainable creation of useful software in a responsible way.

People

Everyone who is concerned with coding, testing, code quality, software craftsmanship, and/or if …

  • you are passionate about software development and testing.
  • you have already heard of Software Craftsmanship or even already are part of the raising community.
  • you want to share your knowlege, experience and expertise.

Action

The event will be run as a non-profit, low-cost event. It is a joint-effort of all Softwerkskammer groups, the community of all German-speaking Software Craftsmanship groups. It will be much like a retreat. We will be there for 48 hours to collaborate and share ideas. There will be:

  • Highly collaborative interactions
  • A self-organized Open Space, with …
    • hands-on coding sessions
    • sessions focused on discussion
    • interactive talks
    • lots more

Registration

The registration will be done via the Softwerkskammer website. In order to be able to register to SoCraTes, you need to enrol in Softwerkskammer first. Once you are enrolled, you can join a group that we set up for everybody who is interested in participating in SoCraTes:

http://www.softwerkskammer.org/groups/socrates2014

The group members will be notified first about the opening of the registration and anything else that might be of importance.
